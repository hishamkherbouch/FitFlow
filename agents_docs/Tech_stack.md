# Tech Stack & Tools
- **Frontend:** Flutter (Dart) for iOS-first development.
- **Backend:** Supabase (PostgreSQL, Auth, Storage).
- **AI Brain:** Gemini Pro 1.5 via API.
- **Nutrition Data:** Open Food Facts API (Free).
- **Environment:** Cursor Pro for code generation.

## Key Logic
- **Context Engine:** The app must fetch 7-30 days of JSON logs from Supabase to feed the AI Coach.
